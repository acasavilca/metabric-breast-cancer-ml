# Notebooks
1. 1_preprocessing_EDA.ipynb: This script is responsible for EDA and preprocessing the dataset, and generated a training.csv and test.csv files for downstream ML models. RUN THIS SCRIPT FIRST
2. 2_logisticRegression.ipynb: Notebook for building a hyperparameter tuned logistic regression model with metrics and model explainability
3. 3_GBDT.ipynb: Notebook for building a hyperparameter tuned gradient boosting decision trees model with metrics and model explainability
4. 4_DT.ipynb: Notebook for building a hyperparameter tuned decision tree model with metrics and model explainability
5. 5_SVM.ipynb: Notebook for building a hyperparameter tuned support vector machine model with metrics and model explainability
6. 6_RF.ipynb: Notebook for building a hyperparametered tuned random forest model with metrics and model explainability
